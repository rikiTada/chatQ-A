目標設定の方法
  • 目標を明確にする - 目標が明確であることは、達成するための道筋を見つける上で重要です。
  • SMART原則に従う - 目標はSpecific（具体的）、Measurable（計測可能）、Achievable（達成可能）、Relevant（関連する）、Time-bound（期限付き）であるべきです。
  • 達成可能なスケジュールを作成する - 目標の達成に向けたステップとタイムフレームを設定します。
  • 追跡する - 進捗状況を定期的に追跡することで、目標の達成に向けた努力を継続することができます。
  • 調整する - 目標の達成に向けた計画が必要に応じて調整されることを想定しておきます。
js数字操作メソッド
メソッド 説明 例
toFixed(n) 数値を指定された小数点以下の桁数で丸めた文字列を返す。 let x = 3.14159; x.toFixed(2); // "3.14"
toPrecision(n) 数値を指定された桁数で丸めた文字列を返す。 let x = 123.456; x.toPrecision(4); // "123.5"
Math.round(num) 数値を四捨五入して整数に変換する。 Math.round(3.14159); // 3
Math.floor(num) 数値を切り捨てて整数に変換する。 Math.floor(3.99999); // 3
Math.ceil(num) 数値を切り上げて整数に変換する。 Math.ceil(3.00001); // 4
Math.trunc(num) 数値の小数点以下を切り捨てて整数に変換する。 Math.trunc(3.99999); // 3
